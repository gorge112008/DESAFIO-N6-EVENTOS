BIENVENIDO AL SIMULADOR INTERACTIVO DE VENTA DE COMIDA RÁPIDA: ***DELIPASO***

Come todo lo que puedas imaginar a un solo clic!!!

DESAFÍO ENTREGABLE: INCORPORAR EVENTOS

El proyecto se basa en la creación de un Simulador Interactivo de Venta de comida rápida.

Premisa: Algoritmo de simulación de un negocio de venta de comida rápida con la creación del menú de comida como objetos con distintas propiedades mediante una función constructora, agregando los menús dentro de un array y aplicando un filtro para obtener nuevos arrays con variados conjuntos de menús.
Definiendo métodos y funciones se ejecutará el correcto pedido o cancelación de cada menú e imprimirá un reporte detallado de cada menú.
Actualización:

Se incorporó el uso de eventos que permitirá manipular el DOM para trabajar directamente en el HTML.
En base a una barra de navegación se podrá ver todo el menú disponible.
Se podra añadir y cancelar pedidos.
Al finalizar el pedido se podra observar al detalle el pedido realizado y se implementará un botón de finalización.
Este botón finalizará la simulación, permitiendo solo mostrar el inicio y el reporte final.
